Takes a gesture LaGeR string and returns the closest matching subscribed gesture.

If no match is found below a certain distance threshold, it indicates it by toggling a Boolean parameter.

Another Boolean parameter determines whether or not to invoke lager_viewer to draw the input and matching gestures on screen.
